# GarbageHub

Create a `_secrets.php` file in the root directory with the following content:

```
<?php

$db_host = "YOUR-DB-HOST";
$db_user = "YOUR-DB-USER";
$db_pass = "YOUR-DB-PASSWORD";
$db_name = "YOUR-DB-NAME";
```

Requirements:

- PHP 7.4 or higher
- MySQL 5.7 or higher
- Apache or Nginx
- git
